# Execute Script
Copy this line and hit enter on the Termux terminal.
```
curl https://raw.githubusercontent.com/johnsmithgit143/termux-setup/main/termux-setup.sh -o termux-setup.sh &&
chmod +x termux-setup.sh && ./termux-setup.sh
```
